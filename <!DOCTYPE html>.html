a)When HTML5 has become a standard? What were the main reasons for developing new standard?

HTML5 became a standard on October 28, 2014. It was developed to address the need for enhanced multimedia support, improved mobile device compatibility, better semantics and accessibility, offline capabilities, enhanced performance, and standardization for interoperability across different browsers and platforms.

b)What is a difference between W3C HTML5 and WHATWG HTML Living Standard?

The main difference between W3C HTML5 and WHATWG HTML Living Standard lies in their update processes. W3C HTML5 follows a traditional standardization model with periodic updates, while WHATWG HTML Living Standard is continuously updated to reflect ongoing developments in web technology.

c)What is the minimal HTML page code?

This code includes the basic structure of an HTML document:

1. DOCTYPE html: Declares the document type and version of HTML being used.

2. html: The root element of the HTML document.

3. head: Contains metadata about the document, such as the title.

4. title: Defines the title of the HTML document, which appears in the browser's title bar or tab.

5. body: Contains the visible content of the HTML document, such as text, images, links, etc.

d)Comment on semantic and presentational markup. Why presentational markup is considered deprecated now?

Semantic markup provides meaningful structure to content, aiding accessibility and search engine optimization. Presentational markup, focused on style, is deprecated as it hinders accessibility and complicates maintenance.

e)How you can check browser support for particular tag or attribute?

To check browser support for a specific HTML tag or attribute, visit websites like MDN Web Docs or Can I Use. These platforms offer detailed information on the compatibility of various web technologies across different browsers, enabling developers to make informed decisions about feature usage.

f)What are the audio and video formats, officially supported by HTML standard? What happens if you use other format?

The officially supported audio and video formats in HTML are:

1. Audio: The supported audio formats are MP3, Ogg Vorbis, and WAV.

2. Video: The supported video formats are MP4, WebM, and Ogg Theora.

If you use a format other than those officially supported, the behavior may vary depending on the browser. Some browsers may attempt to play the content using built-in codecs or plugins, while others may fail to render the content altogether. It's important to ensure compatibility with the supported formats to ensure consistent playback across different browsers and devices.
